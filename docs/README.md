# Documentation

This directory contains all documentation for the Vuhitra CLI project.

## Directory Structure

### `architecture/`
Architectural documentation, design decisions, and technical specifications:
- System architecture diagrams
- Service interactions and data flow
- Technology stack rationale
- Infrastructure design

### `user-guide/`
End-user documentation and guides:
- Installation instructions
- Configuration guide
- Command reference
- Tutorials and how-to guides
- Troubleshooting

### `examples/`
Working examples and use cases:
- Sample workflows
- Code examples
- Configuration examples
- Best practices

## Documentation Standards

- Use Markdown format (.md files)
- Include clear headings and sections
- Add code examples where applicable
- Keep documentation up-to-date with code changes
- Include diagrams using Mermaid or ASCII art

## Contributing to Documentation

When adding new features:
1. Update relevant user guide sections
2. Add examples if applicable
3. Update architecture docs for significant changes
4. Keep README.md in sync
